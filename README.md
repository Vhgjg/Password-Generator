#Password generator

A simple Password generator tools to create strong password for your account.

##Features

- Create a password ranging from 12 to 36 words.
- Can create a password with UpperCase, LowerCase, numbers, or symbols. Can be selected using checklist.
- Password can be Copy to Clipboard.
- Delete and restart generated password.

##Upcoming feature
- tester of the strength and weakness of a password
- and more....

##Getting Started
git clone
cd
npm install
